# Media Watchlist

A responsive web application built to track and manage personal media collections, including films, soundtracks, and audiobooks. 

## Tech Stack
- **Framework:** React + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel

## Live Demo
https://media-watchlist-z4b4.vercel.app/
## Running Locally

To run this project on your local machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/Lgome3z/media-watchlist.git
2. Install the dependencies and run:
Bash
npm install
Start the local development server:
Bash
npm run dev
